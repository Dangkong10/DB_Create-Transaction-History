# 거래명세서 입력 시스템 - 사용 가이드

## 📱 앱 소개

**거래명세서 입력 시스템**은 창고형 매장에서 거래처별 출고 내역을 빠르고 정확하게 입력할 수 있도록 설계된 태블릿 최적화 모바일 앱입니다.

### 주요 특징

- ✅ **초성 검색**: 'ㅎㅂ' 입력만으로 '해비치' 검색 가능
- ✅ **다중 품목 입력**: 한 거래처의 여러 품목을 한 번에 입력
- ✅ **큰 글씨 및 터치 영역**: 고령 직원도 쉽게 사용 가능
- ✅ **빠른 입력 흐름**: 최소한의 터치로 입력 완료
- ✅ **햅틱 피드백**: 버튼 클릭 시 진동 피드백으로 명확한 반응

## 🚀 시작하기

### 1. 앱 설치

#### Expo Go 앱 사용 (개발/테스트용)
1. 태블릿에서 App Store(iOS) 또는 Play Store(Android)에서 **Expo Go** 앱 다운로드
2. QR 코드 스캔하여 앱 실행

#### 독립 실행 앱 설치 (배포용)
1. 관리자로부터 받은 앱 설치 파일(.apk 또는 .ipa) 다운로드
2. 태블릿에 설치

---

## 📖 사용 방법

### 기본 입력 흐름

#### 1단계: 거래처 선택
1. 화면 상단의 **"거래처 이름"** 입력창 클릭
2. 거래처 이름 또는 초성 입력
   - 예: '해비치' 검색 시 → **'ㅎㅂ'** 또는 **'해비'** 입력
3. 자동완성 목록에서 원하는 거래처 선택

**💡 팁:** 초성 검색을 사용하면 더 빠르게 찾을 수 있습니다!

#### 2단계: 품목 입력
1. **"품목명 입력"** 칸에 품목 이름 입력
2. **"수량"** 칸에 수량 입력 (기본값: 1)
3. 추가 품목이 있는 경우:
   - **"+ 품목 추가"** 버튼 클릭
   - 새로운 품목명/수량 입력

**💡 팁:** 품목을 잘못 추가한 경우 각 품목 오른쪽의 **"삭제"** 버튼으로 제거 가능

#### 3단계: 저장
1. 모든 품목 입력 완료 후 화면 하단의 **"저장하기"** 버튼 클릭
2. 저장 완료 메시지 확인
3. 자동으로 입력 화면 초기화

## 🔧 주요 기능

### 초성 검색

거래처 이름의 초성만 입력해도 검색 가능합니다.

**예시:**
- '해비치' → **ㅎㅂ** 또는 **ㅎㅂㅊ**
- '한올' → **ㅎㅇ**
- '동방모사' → **ㄷㅂ** 또는 **ㄷㅂㅁㅅ**

### 다중 품목 입력

한 거래처가 여러 물건을 가져갈 때 한 번에 입력할 수 있습니다.

**예시:**
```
거래처: 해비치
품목 1: 18색사 x 1
품목 2: 24색사 x 2
품목 3: 바늘 x 1
```

### 취소 및 반품

반품이나 취소 건을 입력할 때는 **"취소 및 반품"** 체크박스를 선택하세요.

## ❓ 자주 묻는 질문 (FAQ)

### Q1. 거래처가 검색되지 않아요
**A:** 다음을 확인해주세요:
- 초성이 정확한지 확인 (예: '해비치' → 'ㅎㅂ')
- 거래처 이름의 일부만 입력해도 검색됩니다
- 거래처 목록에 없는 경우 관리자에게 문의

### Q2. 품목을 잘못 입력했어요
**A:** 다음 방법으로 수정할 수 있습니다:
- 저장하기 전: 각 품목의 **"삭제"** 버튼으로 제거 후 다시 입력
- 저장한 후: 취소/반품 체크박스를 선택하여 취소 건으로 입력 후, 올바른 내용으로 다시 입력

### Q3. 저장한 내역은 어디서 확인하나요?
**A:** 현재 버전은 로컬 저장만 지원합니다. 저장된 내역은 앱의 콘솔 로그에서 확인할 수 있습니다. 향후 구글 시트 연동 기능이 추가될 예정입니다.

### Q4. 앱이 느려요
**A:** 다음을 시도해보세요:
- 태블릿 재시작
- 앱 재시작
- 백그라운드에서 실행 중인 다른 앱 종료

## 🎯 사용 팁

### 빠른 입력을 위한 팁

1. **초성 검색 활용**: 거래처 이름을 전부 입력하지 말고 초성만 입력하세요
2. **자주 사용하는 거래처 암기**: 자주 거래하는 곳의 초성을 외우면 더 빠릅니다
3. **품목 추가 미리 준비**: 여러 품목을 가져가는 경우 미리 품목 추가 버튼을 눌러두세요

### 오류 방지 팁

1. **저장 전 확인**: 저장하기 전에 거래처와 품목이 올바른지 확인하세요
2. **수량 확인**: 수량이 1 이상인지 확인하세요
3. **취소/반품 체크**: 반품 건은 반드시 체크박스를 선택하세요

## 🛠️ 문제 해결

### 앱이 실행되지 않아요
1. 태블릿 재시작
2. 앱 재설치
3. 관리자에게 문의

### 입력한 내용이 사라졌어요
1. 저장하기 버튼을 눌렀는지 확인
2. 앱이 백그라운드로 전환되면 입력 내용이 초기화될 수 있습니다
3. 저장 완료 메시지를 확인하세요

### 키보드가 화면을 가려요
1. 입력창을 위로 스크롤하세요
2. 키보드 닫기 버튼(완료/Done) 클릭
3. 화면 빈 공간 터치
