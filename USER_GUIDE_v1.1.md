# 거래명세서 입력 시스템 v1.1 - 사용 가이드

## 📱 앱 소개

**거래명세서 입력 시스템**은 창고형 매장에서 거래처별 출고 내역을 빠르고 정확하게 입력할 수 있도록 설계된 태블릿 최적화 모바일 앱입니다.

### v1.1 주요 업데이트

- ✅ **실제 데이터 반영**: 92개 거래처, 126개 제품 (여름실/겨울실/부자재)
- ✅ **별칭 검색**: '켈리' 또는 '7조' 모두 검색 가능
- ✅ **데이터 관리**: 거래처 및 제품 추가/삭제 기능
- ✅ **카테고리 분류**: 제품을 여름실/겨울실/부자재로 분류

---

## 🚀 시작하기

### 1. 앱 설치

#### Expo Go 앱 사용 (개발/테스트용)
1. 태블릿에서 App Store(iOS) 또는 Play Store(Android)에서 **Expo Go** 앱 다운로드
2. QR 코드 스캔하여 앱 실행

#### 독립 실행 앱 설치 (배포용)
1. 관리자로부터 받은 앱 설치 파일(.apk 또는 .ipa) 다운로드
2. 태블릿에 설치

## 📖 사용 방법

### 화면 구성

앱은 하단에 2개의 탭으로 구성되어 있습니다:
- **입력**: 거래명세서 입력 화면
- **관리**: 거래처 및 제품 관리 화면

---

## 📝 거래명세서 입력

### 기본 입력 흐름

#### 1단계: 거래처 선택
1. **"거래처 이름"** 입력창 클릭
2. 거래처 이름 또는 초성 입력
   - 예: '해비치' 검색 시 → **'ㅎㅂ'** 또는 **'해비'** 입력
   - 별칭 검색: '켈리케이(Kelly.K)' 검색 시 → **'켈리'** 또는 **'7조'** 입력
3. 자동완성 목록에서 원하는 거래처 선택

**💡 팁:** 초성 검색을 사용하면 더 빠르게 찾을 수 있습니다!

#### 2단계: 품목 입력
1. **"품목명 입력"** 칸에 품목 이름 입력
2. **"수량"** 칸에 수량 입력 (기본값: 1)
3. 추가 품목이 있는 경우:
   - **"+ 품목 추가"** 버튼 클릭
   - 새로운 품목명/수량 입력

**💡 팁:** 품목을 잘못 추가한 경우 각 품목 오른쪽의 **"삭제"** 버튼으로 제거 가능

#### 3단계: 저장
1. 모든 품목 입력 완료 후 화면 하단의 **"저장하기"** 버튼 클릭
2. 저장 완료 메시지 확인
3. 자동으로 입력 화면 초기화

## 🔧 주요 기능

### 초성 검색

거래처 이름의 초성만 입력해도 검색 가능합니다.

**예시:**
- '해비치' → **ㅎㅂ** 또는 **ㅎㅂㅊ**
- '한올' → **ㅎㅇ**
- '태광섬유' → **ㅌㄱ** 또는 **ㅌㄱㅅㅇ**

### 별칭 검색 (NEW!)

일부 거래처는 여러 이름으로 검색할 수 있습니다.

**예시:**
- '켈리케이(Kelly.K)' = '7조'
  - **'켈리'** 또는 **'7조'** 모두 검색 가능
- '태광섬유' = '호남사'
  - **'태광'** 또는 **'호남'** 모두 검색 가능
- '실바구니' = '정읍사'
  - **'실바'** 또는 **'정읍'** 모두 검색 가능

### 다중 품목 입력

한 거래처가 여러 물건을 가져갈 때 한 번에 입력할 수 있습니다.

**예시:**
```
거래처: 해비치
품목 1: 18색사 x 1
품목 2: 24색사 x 2
품목 3: 실리콘코 x 1
```

### 취소 및 반품

반품이나 취소 건을 입력할 때는 **"취소 및 반품"** 체크박스를 선택하세요.

## 🛠️ 데이터 관리

### 거래처 관리

**관리** 탭에서 거래처를 추가하거나 삭제할 수 있습니다.

#### 거래처 추가
1. **관리** 탭 클릭
2. **거래처** 탭 선택 (기본값)
3. "거래처 추가" 폼에서:
   - **거래처 이름** 입력
   - **별칭** 입력 (선택 사항, 쉼표로 구분)
   - 예: "켈리케이(Kelly.K)" 이름에 "7조, 켈리" 별칭 추가
4. **추가하기** 버튼 클릭

#### 거래처 삭제
1. 거래처 목록에서 삭제할 거래처 찾기
2. 오른쪽 **삭제** 버튼 클릭
3. 확인 대화상자에서 **삭제** 선택

### 제품 관리

**관리** 탭에서 제품을 추가하거나 삭제할 수 있습니다.

#### 제품 추가
1. **관리** 탭 클릭
2. **제품** 탭 선택
3. "제품 추가" 폼에서:
   - **제품명** 입력
   - **카테고리** 선택 (여름실/겨울실/부자재)
   - **별칭** 입력 (선택 사항)
4. **추가하기** 버튼 클릭

#### 제품 삭제
1. 제품 목록에서 삭제할 제품 찾기
2. 오른쪽 **삭제** 버튼 클릭
3. 확인 대화상자에서 **삭제** 선택

## ❓ 자주 묻는 질문 (FAQ)

### Q1. 거래처가 검색되지 않아요
**A:** 다음을 확인해주세요:
- 초성이 정확한지 확인 (예: '해비치' → 'ㅎㅂ')
- 별칭으로도 검색해보세요 (예: '켈리케이' → '7조')
- 거래처 목록에 없는 경우 **관리** 탭에서 추가

### Q2. 거래처를 추가하고 싶어요
**A:** 
1. **관리** 탭 클릭
2. **거래처** 탭에서 "거래처 추가" 폼 작성
3. 별칭이 있는 경우 쉼표로 구분하여 입력 (예: "7조, 켈리")

### Q3. 제품을 추가하고 싶어요
**A:**
1. **관리** 탭 클릭
2. **제품** 탭 선택
3. "제품 추가" 폼에서 제품명과 카테고리 선택

### Q4. 품목을 잘못 입력했어요
**A:** 다음 방법으로 수정할 수 있습니다:
- 저장하기 전: 각 품목의 **"삭제"** 버튼으로 제거 후 다시 입력
- 저장한 후: 취소/반품 체크박스를 선택하여 취소 건으로 입력 후, 올바른 내용으로 다시 입력

### Q5. 저장한 내역은 어디서 확인하나요?
**A:** 현재 버전은 로컬 저장만 지원합니다. 저장된 내역은 앱의 콘솔 로그에서 확인할 수 있습니다. 향후 구글 시트 연동 기능이 추가될 예정입니다.

## 🎯 사용 팁

### 빠른 입력을 위한 팁

1. **초성 검색 활용**: 거래처 이름을 전부 입력하지 말고 초성만 입력하세요
2. **별칭 활용**: 긴 이름 대신 짧은 별칭으로 검색하세요 (예: '켈리케이' → '7조')
3. **자주 사용하는 거래처 암기**: 자주 거래하는 곳의 초성을 외우면 더 빠릅니다
4. **품목 추가 미리 준비**: 여러 품목을 가져가는 경우 미리 품목 추가 버튼을 눌러두세요

### 오류 방지 팁

1. **저장 전 확인**: 저장하기 전에 거래처와 품목이 올바른지 확인하세요
2. **수량 확인**: 수량이 1 이상인지 확인하세요
3. **취소/반품 체크**: 반품 건은 반드시 체크박스를 선택하세요

## 🛠️ 문제 해결

### 앱이 실행되지 않아요
1. 태블릿 재시작
2. 앱 재설치
3. 관리자에게 문의

### 입력한 내용이 사라졌어요
1. 저장하기 버튼을 눌렀는지 확인
2. 앱이 백그라운드로 전환되면 입력 내용이 초기화될 수 있습니다
3. 저장 완료 메시지를 확인하세요

### 키보드가 화면을 가려요
1. 입력창을 위로 스크롤하세요
2. 키보드 닫기 버튼(완료/Done) 클릭
3. 화면 빈 공간 터치
